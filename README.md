# 🏢 VAYA - Sabzevar Services Directory

**اپلیکیشن هوشمند برای یافتن خدمات و کسب‌وکارهای سبزوار**

A complete, production-ready service directory application built with Flutter, Node.js, and PostgreSQL.

---

## 📱 Features

✅ **Flutter App** - Native performance on Android, iOS, Web
✅ **Smart Search** - Fast, powerful search with filters
✅ **Business Directory** - Browse services by category and location
✅ **Admin Dashboard** - Complete management panel
✅ **Web API** - RESTful backend with JWT authentication
✅ **PostgreSQL Database** - Scalable data storage
✅ **Docker Support** - Easy deployment

---

## 🛠️ Tech Stack

### Frontend
- **Flutter 3.x** - Cross-platform UI framework
- **Dart** - Programming language
- **GetX** - State management
- **Dio** - HTTP client
- **Hive** - Local storage

### Backend
- **Node.js + Express.js** - API Server
- **PostgreSQL** - Database
- **JWT** - Authentication
- **bcrypt** - Password hashing

### Admin Panel
- **React.js + TypeScript** - Web dashboard
- **Ant Design** - UI components

---

## 🚀 Build APK

### Prerequisites
```bash
# Install Flutter
flutter --version  # Should be 3.x or higher
```

### Quick Build
```bash
cd flutter_app
flutter pub get
flutter build apk --release
```

**APK Location:** `flutter_app/build/app/outputs/flutter-app-release.apk`
